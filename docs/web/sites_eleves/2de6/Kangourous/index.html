Ou on peut trouvé les kangourous

Les kangourous rouges se trouvent dans la majeure partie de l’Australie aride, préférant les plaines plates et ouvertes. Les Gris de l'Est se trouvent du Cap York à la Tasmanie ; Les Gris occidentaux ont une répartition tout aussi large, de l'Australie occidentale à Victoria.
